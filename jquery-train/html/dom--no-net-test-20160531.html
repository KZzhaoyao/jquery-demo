考试须知,节点外的知识点参照

  1. 本次考试不允许自己添加id和class类
  2. 尽量使用id值选取
  3. 不选取通用的class类作为选择器
  4. 多次操作相同的dom节点时,请通过变量保存dom对象
  5. 使用高效的选择器,减少选择器的使用
  6. 良好的变量名命名 $###
  7. 隐藏div 使用 $("###").hide()
  8. 取值 使用 $("###").html() ; input 节点取值 使用 $("###").val() ; $("###").attr() 可以获得任意节点的属性值
  9. 优雅的使用链式写法
